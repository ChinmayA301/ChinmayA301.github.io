# World geometry

`countries-110m.json` is the `world-atlas` 1:110m TopoJSON distribution derived from Natural Earth Admin 0 country boundaries. It contains the `countries` and merged `land` geometry collections used by the homepage globe.

- Source: https://github.com/topojson/world-atlas
- Data download: https://cdn.jsdelivr.net/npm/world-atlas@2/countries-110m.json
- `world-atlas` license: ISC
- Natural Earth data: public domain
